# Resonance — mood music & visuals

Browser-based mood selector with Web Audio synthesis, dynamics processing, and a canvas visualization (spectrum, waveform, particles).

## Run locally

Open `index.html` in a modern browser (Chrome, Firefox, Safari, Edge), or serve the folder with any static server.

## Controls

- **Play / Pause** — starts generated audio (click is required for audio unlock).
- **Moods** — Chill, Happy, Angry, Melancholy (retunes EQ and patterns).
- **Microphone / audio file** — optional; drives the same visual analyzer.

No build step or dependencies.
